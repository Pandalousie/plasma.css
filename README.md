#Flexbox Grid
Grid Layout using flexbox


## Quick Start

### Install Assemble
```
$ npm install --global assemble

```

### Install Sass and compass
```
gem install sass

gem install sass-globbing

```

### Start Server
```
$ npm install http-server -g

$ http-server -p 1234

```
